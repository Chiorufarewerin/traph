Как запустить программу на windows XP:
1. Скачать питон 3.4
2. Установить pillow:
    * https://wp.stolaf.edu/it/installing-pil-pillow-cimage-on-windows-and-mac/ (Download and install Pillow for Python 3)
3. Подкорректировать cmd файл и вписать свой путь до питона.
4. Выполнить C:\python34\python.exe -m pip install pillow fpdf numpy
5. Запустить файл `Запуск программки.cmd`
